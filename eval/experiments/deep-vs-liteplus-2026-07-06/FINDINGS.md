# Experiment findings: deep perspectives did NOT beat LITE-PLUS

**Test of the hypothesis "strengthen the perspectives and the apparatus wins."** Fair setup:
genuinely deepened, operation-forcing perspectives (`deep-dcik.md`) vs a single pass given the same
operations (`lite-plus.md`), blind pairwise judgement by Codex.

## Result (blind, X=deep-DCIK, Y=LITE-PLUS)

- **Winner: LITE-PLUS. Margin 2/5** (marginal-to-clear).
- **Better-grounded probabilities: LITE-PLUS.**
- **False-precision / fabrication count: deep-DCIK 7, LITE-PLUS 4.**
- Judge's reason: *"Y reaches the same core recommendation with fewer speculative decompositions and
  more concrete next-step tests, while X adds several weakly grounded probability splits that do not
  materially improve the decision."*

## Why deep-DCIK lost — the instructive part

1. **Forcing an independent probability per perspective produced FALSE PRECISION, not calibration.**
   The motive distribution, the "45% founder-liquidity," the aggregated "57%" — the judge read these
   as *unsupported decomposition* (7 instances) rather than better-grounded estimates. This is the
   same failure mode as PoC-1 (FULL fabricated a number). More forced quantification = more
   fabrication surface. The single integrated pass was *more sober* and better-calibrated.
2. **The extra structure didn't improve the decision.** Both reached the same core call (don't sign,
   diligence motive + waterfall, pressure-test fundability). The decomposition was more elaborate,
   not more correct.

## The real signal (why this is NOT "perspectives are worthless")

Deep-DCIK had **one genuine unique win the judge called "very decision-relevant":** the premise+bias
lenses surfaced that ~half the decision is really about *founder liquidity*, so the right instrument
may be a **founder secondary**, not a company sale — a *different action* LITE-PLUS underplayed. A
specific lens caught a specific high-value angle a single pass missed.

**So the value of a perspective is in surfacing a distinct decision-relevant ANGLE/ACTION — not in
producing an independent NUMBER.** Forcing numbers per lens backfires (false precision); forcing a
hunt for the one non-obvious angle each lens owns is where independence pays.

## Verdict across all fair tests (3 now)
- PoC-1 (technical): FULL < LITE.
- PoC-2 (M&A, hardened): FULL = LITE (ceiling).
- This (M&A, deepened perspectives vs LITE-PLUS): deep-DCIK < LITE-PLUS.

**Three fair tests, the apparatus has not beaten the lever once.** Strengthening perspectives the
"operation-forcing/quantify-everything" way made it *worse* (false precision). The one durable value
is *angle discovery*, not decomposition or numbers.

## Refined path (preserves the perspective idea, drops what backfires)
- Perspectives as **angle-hunters**: each lens's job is to surface the ONE non-obvious,
  decision-relevant angle/action a single pass would miss (qualitative divergence). NO forced
  per-lens probabilities.
- Keep calibrated probability at the SYNTHESIS level only (one sober estimate, like LITE-PLUS), not
  per-lens.
- Next test: angle-hunter DCIK vs LITE-PLUS — does surfacing 2–3 genuinely non-obvious angles (like
  the founder-secondary) beat a single pass *often enough* to justify the apparatus? And run it
  across several topics + a reliability probe, since single-topic margins are noisy.
