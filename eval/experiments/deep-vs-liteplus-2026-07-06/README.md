# Experiment: do DEEP perspectives beat LITE-PLUS?

**Question under test (user hypothesis):** the prior evals convicted DCIK on a strawman — thin,
role-played perspectives. If perspectives are genuinely deep and operation-forcing, does the
apparatus beat the simple lever *when the lever is also given the same operations*?

**Topic:** the `acquisition-offer` decision (reused; prior FULL vs LITE tied at the recall ceiling).

## The three things that make this a FAIR test

1. **The baseline is LITE-PLUS, not vanilla LITE.** A single pass explicitly prompted with the same
   operations (reference class + base rate, calibrated probability, premortem, disconfirmation).
   Beating vanilla LITE would prove nothing (deep beats shallow — trivially). The question is whether
   deep-DCIK beats a *single deep pass*.
2. **The perspectives are strengthened the RIGHT way** — into *operation-forcing, independence-
   generating* protocols (each yields an independent estimate + a typed artifact), not longer prose.
   Prose depth stays promptable; forced independent estimation is the only thing a single pass
   structurally can't replicate (Self-MoA logic).
3. **Non-saturating instrument.** Recall saturated last time (both 10/10). Here the primary judge is
   **blind pairwise preference** (Codex: which decision memo is better, by what margin, why) — which
   does not ceiling — plus a regression/fabrication check.

## Arms
- `lite-plus.md` — one strong pass + the operations, prompted explicitly.
- `deep-dcik.md` — the 6 strengthened perspectives (`deepened-perspectives.md`) each run as an
  independent analysis producing its own estimate, then aggregated into a decision.

## Honest limits (stated up front)
- Single topic, single blind judge, author-generated arms. This tests the *hypothesis* (deep beats
  LITE-plus on one high-stakes decision); it is not the confirmatory study.
- Pairwise preference is a judgement, not ground truth. A forecast-type question with resolvable
  outcomes (Brier) is the next instrument; this decision question has no clean resolution.
- If deep-DCIK does NOT beat LITE-plus here, strengthening the other ~170 perspectives is very
  unlikely to change the verdict — that is the point of doing the minimal version first.
