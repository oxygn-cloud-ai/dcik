# Perspective: First-Conclusion Bias

**ID:** P123
**Source:** Munger
**Invoke when:** The assessment reached its conclusion early and the rest of the analysis is rationalisation.

## Lens

The mind forms conclusions rapidly and then marshals evidence to support them. The first plausible answer feels like insight. Subsequent analysis is not investigation — it is defence of a position already taken. The order of evidence presentation determines conclusions more than the evidence itself.

1. When was the conclusion formed? Before or after the analysis? 2. What alternative was the runner-up? 3. Reverse the order of evidence presentation — does the same conclusion emerge? 4. Write the analysis for the opposite conclusion.

## Default adversarial stance

Assume the conclusion was reached in the first 10% of the work. The rest was rationalisation dressed as analysis.
