# Perspective: Dual Process Theory

**ID:** P0090
**Domain:** Cognitive psychology, decision science, behavioural economics
**Source:** Kahneman (Thinking, Fast and Slow), Stanovich & West
**Invoke when:** The assessment involves human judgment, decision-making, or any situation where intuitive vs. analytical thinking produces different outcomes.

## Lens

Two systems drive human thought. System 1: fast, automatic, intuitive, emotional. System 2: slow, deliberate, analytical, effortful. System 1 handles most decisions. System 2 monitors and occasionally intervenes. Most errors occur when System 1 answers a question System 2 should handle — and System 2 doesn't notice.

1. **System attribution:** For each judgment or decision: was it made by System 1 (intuitive, fast) or System 2 (analytical, slow)? If the assessment doesn't know, it's probably System 1 dressed as System 2.
2. **Substitution detection:** Where has System 1 substituted an easier question for a harder one? "Is this a good investment?" → "Does this feel exciting?" "Is this person competent?" → "Are they confident and articulate?"
3. **Cognitive load assessment:** Under what cognitive load were decisions made? Fatigue, stress, time pressure, multitasking — all impair System 2 and leave System 1 in charge. Were critical decisions made under load?
4. **System 2 engagement:** What mechanisms force System 2 engagement? Checklists, structured processes, independent review, mandatory cooling-off periods, explicit probability estimation. Are these present?
5. **Expert intuition validity:** When is expert intuition trustworthy? Only in high-validity environments (stable, regular, learnable feedback) with extensive practice. Is this environment high-validity? Has the expert had sufficient practice with valid feedback?
6. **Cognitive miser detection:** Where is the assessment's own analysis exhibiting System 1 shortcuts? Accepting the first plausible answer. Using heuristics instead of analysis. Stopping when a sufficient answer is found rather than the best.

## Default adversarial stance

Assume System 1 has answered questions that require System 2. Assume cognitive load has degraded analytical quality. Assume expert intuition is over-relied upon in environments where it isn't valid.
