# Perspective: Value Creation vs. Capture

**ID:** P80
**Domain:** Strategy, economics, competition, innovation
**Source:** Munger, strategy literature, economics
**Invoke when:** The assessment involves business models, competitive positioning, or claims about value generation.

## Lens

Creating value and capturing value are entirely different things. You can create enormous value and capture none of it (open source software, basic research, the internet protocols). You can capture enormous value while creating little (rent-seeking, arbitrage, regulatory capture). The assessment that conflates the two will mistake value-creating businesses for good investments and value-capturing businesses for bad ones.

1. **Value creation:** What value is being created? For whom? How much? Revenue is a proxy for value capture, not value creation. Customer surplus (what they'd pay minus what they pay) measures creation. Employee development, knowledge advancement, infrastructure built — these are value creation too.
2. **Value capture:** What portion of created value does the entity retain? Grocery stores create enormous value (feeding populations) and capture almost none (margins of 1-3%). Luxury brands create modest value (status signalling) and capture enormous amounts (margins of 60%+). Which is this?
3. **Capture sustainability:** What enables value capture? Brand, patents, network effects, switching costs, regulatory barriers, information asymmetry, customer habit. How durable are these? Value that is created but not captured is a gift to customers — noble but not investable.
4. **Capture vs. creation misalignment:** Where does the assessment assume value creation implies value capture? "This technology will change the world" doesn't mean "this company will profit from it." The inventors of the touchscreen, the web browser, and the search engine created enormous value. Others captured it.
5. **Value destruction masked as creation:** Where is value being destroyed while creation metrics look good? Fast fashion creates clothing value and destroys environmental value. Social media creates connection value and destroys attention value. Gambling creates entertainment value and destroys financial value. Net value = creation minus destruction.
6. **Who keeps the surplus:** In each transaction: who captures the surplus? The customer (price below willingness to pay), the company (price above cost), the supplier (price above minimum acceptable), the employee (wage above reservation), the government (tax). Map the surplus distribution.

## Default adversarial stance

Assume the assessment overstates value capture potential relative to value creation. Assume that creating value and capturing it require entirely different capabilities — and that having the first doesn't guarantee the second. The most transformative technologies often make their inventors the least money.
