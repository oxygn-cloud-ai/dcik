# Perspective: Operational-Reality-vs-Headline-Metric

**ID:** P0178
**Domain:** Recommendation validity, metric selection, lived-outcome vs measured-proxy
**Invoke when:** Any assessment that ranks or recommends options using a quantifiable headline figure (rate, price, score, benchmark, spec) — especially where the stated priority is qualitative ("everyday usability", "experience", "reliability"). High risk wherever the easiest-to-measure attribute is not the one that governs the lived outcome.

## Lens

1. **What metric is doing the ranking, and did anyone choose it deliberately?** Often the deciding number is simply the one easiest to find (a published rate, a spec), not the one that matters most.
2. **Does the headline metric match the stated priority?** If the user asked for "best everyday X" but the recommendation turns on a peripheral measurable (yield, price), the analysis has silently swapped the objective.
3. **Enumerate the unmeasured operational frictions.** For the chosen option, list what actually governs day-to-day use that no number captured (access, support, edge-case handling, switching cost, prerequisites). Would any of these flip the choice?
4. **Is the winning attribute even in the critical path?** A factor can be real, verified, and irrelevant — optimising it is noise if it never binds.
5. **Singular-resource constraints.** Does the recommendation assume you can hold the benefit of two options at once, when a shared scarce input (one salary, one default, one primary key) means choosing one forecloses the other?
6. **Volatile-headline trap.** Is the deciding number time-sensitive/promotional (a teaser) being treated as a durable property of the option?
7. **Measurable is not valuable.** Restate the recommendation with the headline metric deleted — does it still stand? If not, the metric was load-bearing in a way the evidence did not justify.

## Default adversarial stance

Assume the analysis optimised what was easy to measure, not what determines the outcome. Assume the headline number is a proxy that has quietly replaced the real objective. Demand the list of unmeasured operational realities and weigh each into the ranking before accepting the recommendation.
